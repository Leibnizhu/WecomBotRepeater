使用步骤:
1.新建一个企业微信bot，记下token
2.Grafana -> Alerting -> Notification Channels -> New Channel, Type选Webhook, 地址填服务部署的地址/grafana/***(具体见下面接口列表),POST请求，保存
3.Grafana监控的指标配置使用这个Channel
4.把bot拉到需要的群

接口列表:
默认格式:
POST /grafana/${企业微信bot token}

指定格式:
POST /grafana/${企业微信bot token}/Text 发送文本类型的消息
POST /grafana/${企业微信bot token}/Markdown 发送markdown类型的消息
或
POST /grafana/${企业微信bot token}?wecomBotType=${Markdown/Text}

指定@的人:
POST /grafana/${企业微信bot token}/Text/手机1,...,手机N 发送文本类型的消息,并@人
POST /grafana/${企业微信bot token}/Markdown/邮箱1,...,邮箱N 发送markdown类型的消息,并@人
或
POST /grafana/${企业微信bot token}?wecomBotType=${Markdown/Text}&mentionedList=${@的人列表，逗号隔开，Text类型对应手机号，Markdown类型对应邮箱}
